Clasificación en CIFAR-10 con MLP
Este proyecto implementa una red neuronal densa optimizada para clasificar imágenes.

Instrucciones de ejecución
Abrir el archivo .ipynb incluido en este repositorio.

Ejecutar todas las celdas en Google Colab.

En caso de error 503 en la descarga del dataset, subir manualmente el archivo cifar-10-python.tar.gz proporcionado al entorno de Colab.
